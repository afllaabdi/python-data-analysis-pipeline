# Python Data Analysis Pipeline

Project ini menunjukkan proses dasar dalam data science workflow:
data cleaning → data analysis → business insight extraction menggunakan Python.

## 1. Data Cleaning (List Processing)

Melakukan manipulasi list:

* menambahkan data acak
* filtering berdasarkan index
* sorting dan penghapusan nilai ekstrem

Tujuan: mensimulasikan preprocessing dataset sebelum modeling.

## 2. Academic Grade Analysis (Dictionary & Tuple)

Menganalisis data nilai mahasiswa:

* menghapus missing values
* sorting nilai
* mencari nilai tertinggi
* menghitung rata-rata

Tujuan: memahami proses data validation & statistics.

## 3. Business Sales Analysis (Pandas DataFrame)

Analisis dataset penjualan:

* descriptive statistics
* revenue calculation
* top performing products
* price segmentation (cheap / medium / expensive)

Tujuan: mengubah data mentah menjadi insight bisnis.

## Insight

Sebagian besar pekerjaan data science bukan membuat model, tetapi membersihkan dan memahami data.
